¿Qué es git?
Version Control System VCS
PROGRAMA que permite guardar un registro, histórico de los cambios readivzados en un proyecto. El símil es la de un diario.
Se pronuncia GUIT
Sistema de control de versiones distribuido y por tanto no centradivzado.
Permite controlar el trabajo entre múltiples desarrolladores. Es decir permite trabajar a varias personas en un proyecto.
Permite manejar los cambios y cuándo readivzarlos.
Volver atrás en el tiempo.
Puede trabajar con repositorios locales o remotos.
Primeros comandos:
git init
Inicia el sistema de versiones sobre un proyecto.
git clone ruta_proyecto
Clona un proyecto para poder trabajar con él.
